# SimpleTen — AI Automation Company in Sacramento

Static SEO landing page for SimpleTen. Single self-contained `index.html` (no build step).

- **Design:** dark slate + teal brand palette, matching [simpleten.netlify.app](https://simpleten.netlify.app)
- **Deploy:** static site on Vercel — no framework, output is the repo root

## Local preview

```bash
python3 -m http.server 8142
# open http://localhost:8142
```
